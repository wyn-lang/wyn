# Contributing

## Development

```bash
make              # build compiler
make test         # run tests
./wyn run <file>  # test changes
```

## Testing

All changes need tests. Add tests in `tests/` directory.

## Code Style

- Keep functions small
- Use clear names
- Add comments for complex logic
- Follow existing patterns

## Pull Requests

- Include tests
- Update CHANGELOG.md
- Ensure all tests pass
