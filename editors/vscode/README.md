# Wyn Language Support for VSCode

Basic syntax highlighting for the Wyn programming language.

## Features

- Syntax highlighting for keywords: `let`, `var`, `fn`, `for`, `if`, `match`
- String and number highlighting
- Comment highlighting (# line comments)
- Function call highlighting

## Installation

1. Copy this extension folder to your VSCode extensions directory:
   - **Windows**: `%USERPROFILE%\.vscode\extensions\`
   - **macOS**: `~/.vscode/extensions/`
   - **Linux**: `~/.vscode/extensions/`

2. Restart VSCode

3. Open any `.wyn` file to see syntax highlighting

## Development

To test changes:
1. Open this folder in VSCode
2. Press `F5` to launch Extension Development Host
3. Open a `.wyn` file in the new window
