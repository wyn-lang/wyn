Install Wyn

Fast, simple, universal programming language for all platforms.

Quick Install

Install via curl (Recommended)

curl -fsSL https://wynlang.com/install.sh | sh
What this does:
  • Detects your platform (macOS/Linux) and architecture (ARM64/x86_64)
  • Downloads the latest release from GitHub
  • Installs to /usr/local/bin
  • May require sudo password

Alternative: Homebrew (Coming Soon)

brew install wyn-lang/tap/wyn

Install via PowerShell (Recommended)

irm wynlang.com/install.ps1 | iex
What this does:
  • Downloads the latest Windows release from GitHub
  • Installs to %LOCALAPPDATA%\Wyn
  • Adds to your PATH automatically
  • Restart terminal after installation

Alternative: Chocolatey (Coming Soon)

choco install wyn

Direct Downloads

Download the binary for your platform:

macOS ARM64
Apple Silicon (M1/M2/M3) macOS x86_64
Intel Macs Linux x86_64
Most Linux systems Linux ARM64
ARM servers/devices Windows x86_64
64-bit Windows

Manual Installation Steps

  1. Download the appropriate archive for your platform
  2. Extract: tar -xzf wyn-*.tar.gz
  3. Move binary to PATH:
    • Unix: sudo mv release/wyn /usr/local/bin/
    • Windows: Add release folder to your PATH
  4. Verify: wyn --version

Verify Installation

wyn --version

Quick Start

echo 'fn main() { print("Hello, Wyn!") }' > hello.wyn && wyn run hello.wyn
